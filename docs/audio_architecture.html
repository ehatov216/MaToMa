⑥ 自律システム — Autonomous / Chaos
🌀 ChaosEngine（Python）
Dejavu パターンによる記憶付きパラメータードリフト。
SC 起動完了と同時に自動スタート。

各パラメーターに 引力点（Attractor） を設定し、
カオスの振れ幅は カオス幅エンベロープ で長期制御。
「完全ランダム」でも「固定スコア」でもない第三の状態。

→ Drone: freq, feedback_amt, shimmer, room → Granular: density, spray, pos, size
⚡ TuringMachine（SC内）
16ステップ・シフトレジスタがリズムパターンを確率的に進化。

prob=0.0 → 完全ループ（ミニマル）
prob=0.5 → 徐々に変化（Autechre的・最も音楽的）
prob=1.0 → 完全ランダム（崩壊）

Rhythmic の5音源（Grain/Klank/FM/Spring/Chaos）に
「どの音源を・いつ・どんな音程で叩くか」を確率的に配分。
