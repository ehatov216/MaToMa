## 制御信号の流れ

```
[あなた]
  シーン切り替え ───────────────────────────────→ Scene Manager
  エフェクト操作 ───────────────────────────────→ Effect Layer（即時）

[Scene Manager / Python]
  引力点 + 揺れ幅の上限 ────────────────────────→ Chaos Engine

[Chaos Engine + Dejavu / Python]
  パラメータードリフト（秒〜分）─────────────────→ 全層（OSC経由）

[TidalCycles]
  ノートイベント + タイミング ──────────────────→ Granular / Rhythmic（カスタムOSC）

[Turing Machine / SC内部]
  パターン記憶 + 確率変異 ──────────────────────→ Rhythmic 発火パターン

[Fb1 Lorenz / SC内部]
  数学的カオス（ミリ秒〜秒）───────────────────→ Drone / Granular 音色変調
```

## レイヤー間の依存関係

```
Drone（Wet）─→ Granularの素材になる
               （霧ごと粒にすることでHecker的密度が生まれる）

Rhythmic ─────→ Space層と独立して動く
               （Tidalのパターンで完全に制御）

Effect層 ─────→ 全音源のミックスに後処理をかける
               （ユーザーがここで音の質感を手で整える）
```

## シーンが変えるもの

```
シーン「深淵」                    シーン「崩壊」
  Space層が支配                    Event層が前に出る
  Drone: 低域・chaos低             Drone: 高域・chaos高
  Granular: 粒が疎                 Granular: 粒が密
  Rhythmic: PROB低（安定ループ）   Rhythmic: PROB高（崩壊）
  Reverb: room大（広大）           Reverb: room小（ドライ）
```
