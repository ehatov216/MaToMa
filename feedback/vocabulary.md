# 音の感想を伝えるための語彙集

フィードバックセッションで「この音をこうしてほしい」と伝えるための言葉リスト。
専門知識がなくても使えるよう、日常語を中心にまとめています。

## 7. 程度・比較の言い方

### 量を調節する
- **もっと**〇〇に → 大きく変えたいとき
- **少し**〇〇に → 小さく変えたいとき
- **ほんの少し**〇〇に → 微調整したいとき
- **かなり**〇〇に → 大胆に変えたいとき

### 比較する
- **さっきより**〇〇 → 直前の状態と比べて
- **最初より**〇〇 → セッション開始時と比べて
- **もう少し**〇〇 → 今より少しだけ

### 組み合わせ例
- 「**もっと**暗くしてほしい」
- 「**少し**リバーブを減らしてほしい」
- 「**さっきより**ざらざら感が増した気がする」
- 「**もう少し**揺れる感じにしてほしい」

## 8. そのまま使えるフレーズ集

### 「こうしてほしい」
- 「もっと〇〇な感じにして」
- 「〇〇を増やしてほしい」
- 「〇〇を減らしてほしい」
- 「〇〇が足りない気がする」
- 「〇〇が多すぎる気がする」

### 「気づいたこと」を伝える
- 「なんか〇〇な感じがする」
- 「さっきより〇〇になった」
- 「〇〇と〇〇が混ざったような感じ」
- 「〇〇みたいなイメージ」（比喩でOK）

### 評価する
- 「いい感じ！」
- 「方向性は合ってる、でも〇〇が少し違う」
- 「変わってない気がする」
- 「なんか違う（どこが違うか言葉にできなくてもOK）」

## 9. MaToMaで特によく使う表現

このプロジェクトは「制御されたカオス」がテーマなので、こんな言葉がよく出てきます。

| 言葉 | 意味 |
|------|------|
| カオス度 | どれくらいランダムで予測不能か |
| 自律性 | システムが自分で動いている感じの度合い |
| 粒（グレイン） | 音の最小単位。米粒ほどの短い音の断片 |
| 揺れ | パラメーターがゆっくり変化している感覚 |
| 空間感 | 音が広がっている感じ（リバーブで変わる） |
| 密度 | 音がどれだけ詰まっているか |

---

*このリストは使いながら育てていくものです。「こういう言葉が足りない」「この言葉の意味が分からない」はいつでも教えてください。*
