# Burialの音楽的特質・制作技法・ハウントロジー

## 概要

William BevanことBurialは、2000年代半ばにHyperdubレーベルから登場。UKガラージ、ダブステップ、ジャングルの残響を、深夜のロンドンという地理的・心理的文脈へと再構築した。その音響美学はAutechreとは対極にあり、「震える個人の足取り」というヒューマニスティックな地平を目指す。

---

## 制作環境の特異性：Sound Forgeという選択

### 非グリッド・手作業による制作
- 使用ソフト：**Sony Sound Forge**（マルチトラックDAWではなく波形編集ソフト）
- グリッド・クオンタイズを一切使わない
- 波形の断片を**1つずつ手作業で配置**
- リズムの正しさは**波形の視覚的形状と聴覚のフィーリング**で判断
- ドラムパターンがプロジェクト上で「**魚の骨（fishbone）**」に見えたとき、それが正しい

### Autechreとの対比
| 項目 | Burial（Sound Forge） | Autechre（Max/MSP） |
|------|----------------------|---------------------|
| リズム同期 | 手作業・目視配置（非グリッド） | アルゴリズム・数学的生成 |
| 制作の動機 | 直感・雰囲気・偶然性の許容 | 構造的整合性・再現性 |
| 制作の性質 | コラージュ・彫刻（肉体的作業） | プログラミング（思考的作業） |
| リズムの美学 | 震える個人の足取り | 超人間的な複雑さ |

### 破壊的編集の美学
- Sound Forgeは基本的に**破壊的編集**（元データに直接上書き）
- 「一度決定したら戻れない」不可逆性が特有の緊張感と一回性を生む
- 内蔵リバーブで音を「洞窟のような」深い残響に沈める
- リバーブは音の背後にある「**空虚**」や「**不在**」を強調するために機能

**MaToMaへの示唆：精密な自動化より、意図的な「ズレ」「不完全性」が人間的な体温を生む。**

## 音響素材：都市のFoleyとゲームの記憶

### 金属音・環境音によるパーカッション
- 標準的なドラムマシンを使わない
- 家の中で録音した「**叩く音・こする音・金属的な衝撃音**」でドラムを構成
- PlayStation『**Metal Gear Solid**』の銃の薬莢が落ちる音などを重要アクセントに使用
- 現実世界の物理的質感を音楽に持ち込む → 具体的な情景を喚起

### 意図的ノイズの混入
- **ビニールノイズ・テープのヒス**を意図的に残す
- クリーンな音を「ニュートラルすぎて意味がない」と拒絶
- ノイズを混入させることで音楽に**歴史的な厚みと幽霊のような気配**を付与

```
「クリーンであることを拒絶する」
→ ノイズ＝時間の蓄積・記憶の証拠
```

### R&Bボーカルの解体とジェンダーの流動性
- BeyoncÃ©, Ciara, Usherなど著名R&Bシンガーの楽曲やYouTubeのカバー動画を素材
- 切り刻み・ピッチ変更で**元のアイデンティティを消去**
- 結果：男女どちらでもない、ジェンダー的に流動的な「声」に変容
- 意味のある文章ではなく「I stay alive」「Come down to us」などの断片的フレーズ
- → **情動のしぶき**として潜在意識に直接訴えかける

## リズムの音響的分析：オフキルター・スウィング

### ガラージからの継承と不安定化
- UKガラージ・2ステップの「跳ねるスウィング感」を継承しつつ**極端に不安定化**
- スネア・ハットが数ミリ秒遅れる／早まる → **機械的グルーヴを排除**
- この「遅延」「ヨレ」が音楽に**歩行・心拍のような人間的生理感覚**を与える

### 空間設計：「霧」と「奥行き」
- マーク・フィッシャー「霧の中にいるような、拡散しているが自分を包み込んでいる感覚」
- レイヤー化された環境音 + 深いリバーブ + 高域カットの「くぐもった音質」
- **クラブのサウンドシステムではなく**、深夜のバス・ヘッドフォンでの内密な体験を前提
- → 全身を震わせる音ではなく、**耳元に寄り添う音**

## ハウントロジー（幽霊学）の位相

### マーク・フィッシャーと「失われた未来」
- 『Burial』『Untrue』を**ハウントロジーの最良の実践例**として位置づけ
- 「1990年代のレイヴ・カルチャーが約束した未来」が実現しなかったことへの喪失感
- Burial自身はそのシーンを直接体験するには若すぎた → 記憶は常に「他者の断片」
- **不在の記憶への哀歌（エレジー）** = 比類なき情緒的深度の源

### ハウントロジーの音楽的現れ

| 概念 | Burialの音楽における現れ | 批評的意味 |
|------|------------------------|-----------|
| ハウントロジー | 過去のボーカル・レコードノイズ・残響 | 実現しなかった未来への哀悼 |
| 場所の喪失 | 空虚なロンドンの街並み・地下鉄の音 | 都市における孤独と疎外 |
| 歴史の終わり | ジャンルの解体と再構築 | 文化的停滞と記憶の循環 |
| ハードコア・コンティニュアム | ガラージやジャングルのエコー | UKダンス音楽の地下伏流水 |

### ビニール・クラックルの象徴性
- 「過去のメディア」が現代のデジタル空間に**亡霊として侵入**している徴候
- レトロな装飾ではなく、歴史的・批評的な意味を持つ音響記号

## 作品の変遷

### 『Untrue』（2007）：ハウントロジーの完成形
- ピッチ変更されたボーカルが前面に
- 深夜ロンドンの情景を最も完璧に体現

### 『Rival Dealer』（2013）：個人的声明への転換
- 「都市の孤独」から、セクシュアリティ・ジェンダー・「自分であることを恐れない」へ
- クリアなスポークン・ワード、解放感のあるシンセ・メロディが突如導入
- 孤独が「連帯」へと変容する瞬間を捉える

### 『Antidawn』（2022）：ビートの消滅とスタシス
- パーカッションがほぼ完全消失
- 風・咳・ライターの着火音・断片的なオルガン・極度加工された声のみ
- 「**ダブステップからダブ（残響）だけを抜き出した結果**」（批評家の評）
- ダンスミュージックの骨格が消え、純粋な「気配」だけが残る

## Burialが示した3つの逸脱（MaToMaへの指針）

1. **技術的逸脱：効率性・正確性の追求からの離脱**
   - Sound Forgeの手作業でデジタルの中に「誤差・ズレ・不完全性」を再導入
   - → 完璧なクオンタイズより、**意図的な揺らぎ**を設計する

2. **形式的逸脱：ダンスミュージックの機能性からの離脱**
   - クラブのフロアではなく、**個人の内省的空間を形作る**音を目指す
   - ボーカルを情動の記号として使う（意味ではなく「気配」）

3. **歴史的逸脱：現在という閉鎖性からの離脱**
   - 過去の残響と現代の虚無感を接続する
   - **失われた未来についての対話**を音楽で行う

## AutechreとBurialの比較要約

| 軸 | Autechre | Burial |
|----|----------|--------|
| 制作哲学 | アルゴリズム、数理的秩序 | 手作業、直感、偶然性 |
| リズム美学 | 機械による反乱・超人間的複雑さ | 震える個人の足取り |
| 音響的質感 | Dry・骨格の提示 | Wet（ただし意図的な霧）・包み込む |
| 匿名性の機能 | 作者排除・純粋な物理的対象として音を提示 | 都市の孤独な放浪者イメージの補強 |
| 情緒的立場 | 冷徹な知性による未知の音響探求 | 傷つきやすい感性による過去と現在の対話 |
| 聴取空間 | コンサートホール・マルチチャネル | 深夜のバス・ヘッドフォン |
