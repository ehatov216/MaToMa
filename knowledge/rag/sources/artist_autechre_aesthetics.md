# Autechreの計算論的美学とアルゴリズム的音楽論

## 概要

マンチェスター出身のショーン・ブース（Sean Booth）とロブ・ブラウン（Rob Brown）によるユニット、Autechreは1987年の結成以来、電子音楽の境界を絶えず拡張し続けてきた。初期のデトロイト・テクノやヒップホップ、エレクトロの影響を色濃く残したメロディックなスタイルから、抽象的かつ複雑なアルゴリズムを用いた実験音響へと劇的な進化を遂げている。

---

## 時代区分と音響的特徴

| 時代区分 | 代表作 | 主要な音響的特徴 | 技術的アプローチ |
|---------|--------|----------------|----------------|
| 第1期（1991-1994） | Incunabula, Amber | メロディック、深遠な残響、エレクトロの影響 | ハードウェア・シーケンサー、アナログシンセ |
| 第2期（1995-1999） | Tri Repetae, LP5 | ドライな質感、工業的ノイズ、リズムの複雑化 | デジタルシンセ、サンプラーの多用、Max/MSPの導入 |
| 第3期（2001-現在） | Confield, Exai | ジェネレーティブ、FM合成、非人間的抽象性 | Max/MSPによる完全自作システム |
| 第4期 | elseq, NTS Sessions | 超長時間、極限のFM合成、パラメータ空間の探索 | 統合されたソフトウェア環境（The System） |

---

## 重要な美学的転換：Wet → Dry

### 第1期：「ウェット（Wet）」な音像
- エコーを多用した残響豊かな空間
- 抒情的なメロディと複雑なリズムが共存
- 「サブマリンのような有機的なエコー」が特徴

### 第2期以降：「ドライ（Dry）」への移行（Tri Repetae, 1995が境界）
- リバーブによる空間演出を抑制
- デジタル特有のクリック音、鋭利なパーカッションを前面に
- 「音そのものの骨格を提示することへの自信」の表れ
- アナログ的な曖昧さへの決別

**MaToMaへの示唆：リバーブを安易に追加しない。乾いた音の骨格を恐れない。**

## 使用機材の変遷

### 初期ハードウェア
- Roland TR-606, MC-202（安価な中古機材）
- Casio SK-1, SK-5（内部チップをショート＝回路曲げによるリング変調・フランジング）
- 「機材の限界を試す」姿勢 → 後の「バグやエラーを音楽として取り込む」思想へ

### ソフトウェア移行（1997年〜）
- **Max/MSP**：MIDIの制限を超えた複雑なモジュレーション、確率論的リズム生成
- **Kyma**：音響合成の実験
- Elektron Machindrum, Monomachine, Nord Modular G2（JJOS等でカスタム）

### 現在：「The System」
- Max/MSP上で統合された完全自作環境
- ショーン・ブース「Max/MSPがあれば必要なものは何でも自分で作れる」

## FM合成の哲学

- ジョン・チャウニングが開発したFM合成を極限まで突き詰める（特にelseq, NTS Sessions）
- **単なるプリセットではなく、オペレーター同士の接続をアルゴリズムで動的に変化させる**
- 複雑な倍音構造を持つ金属的な音、流動的な質感のノイズを生成
- 予測不能な音色の変容がプロセスとして機能

**SuperColliderでの実装示唆：**
```supercollider
// FM比率・変調深度をLFOや確率で動的変化させる
SynthDef(\ae_fm, {
    var mod_ratio = LFNoise1.kr(0.1).range(1.0, 7.0);
    var mod_index = LFNoise2.kr(0.05).range(0.5, 15.0);
    var mod = SinOsc.ar(440 * mod_ratio, 0, 440 * mod_index);
    var car = SinOsc.ar(440 + mod);
    Out.ar(0, car * 0.3);
}).add;
```

## 数理的・学術的手法

### 確率論とストカスティック・プロセス
- 「不規則なリズム」は単純な乱数ではなく、特定の確率分布に基づく
- イアニス・クセナキスの「ストカスティック・ミュージック」と共通の思想
- **マルコフ連鎖シーケンサー**：次の打点が生じる確率を動的に計算

```
あるドラムの打点が、次の打点を打つ確率を動的計算
→ 「確率の勾配」を設計することが作曲行為
```

### フラクタルと自己相似性
- ショーン・ブース「多次元的なパラメータ空間を探索すること」
- 「フラクタルの中を歩き回るようなもの」
- 一つの微小な音の断片が楽曲全体の構造を反映する（スケール非依存の複雑性）
- 楽曲「Glitch」では各セクション比率に黄金比が見出される

### セル・オートマトン
- 格子状のセルが周囲の状態に応じて変化するアルゴリズム
- ジェネレーティブなシーケンス生成の着想源

## 反復の否定：「Flutter」（Anti EP, 1994）

イギリス「刑事司法及び公共秩序法（1994）」が「反復的なビート」を規制対象に。
→ Autechreは「Flutter」で**全編を通じて一度も同じリズムパターンが繰り返されない**プログラムを構築

アルゴリズムによって法律の「反復」定義を無効化。音楽構造自体による社会的抗議。

## ライブパフォーマンス：空間音響の極致

### 暗闇の演出
- ほぼ完全な暗闇の中でライブ
- 視覚を奪うことで純粋な音響体験を強化
- ピエール・シェフェールの「アコースマティック（Acousmatic）」聴取の現代的更新

### マルチチャネル合成（AE_LIVEシリーズ）
- 最大16チャネル以上のスピーカー
- 各音源の3次元座標（x, y, z）をリアルタイム制御
- ガウス関数を用いたロールオフパラメータで物理的音の移動を実現

$$gain_i = e^{-\frac{distance_i^2}{2 \cdot rolloff}}$$

## 学術的位置付け：非人間主義（Inhumanism）

### 20世紀前衛音楽との接続
- イアニス・クセナキス：ストカスティック・ミュージック、大量の音粒子を統計的制御
- カールハインツ・シュトックハウゼン：電子音響の探究
- エドガー・ヴァレーズ：「組織化された音（organized noise）」→ Autechreのリズム解体・音響設計の根底
- ベルナール・パルメジャーニ：ミュジーク・コンクレートへの接近

### 音楽理論の解体
- 「音階」「拍子」を特権視せず、「モジュレーション可能なパラメータの一つ」として等価に扱う
- 「パッチそのものが組成である」（マーク・フェル）
- 最終的なオーディオファイルは「システムが出力した無数の可能性の一つ」に過ぎない

## 3つの功績（MaToMaへの指針として）

1. **機材の主体的再構築**
   - 既製品インターフェースに満足せず、「作曲者自身の思考を直接反映した楽器」としてシステムを構築
   - → MaToMaもUIを通じて音楽的意図を直接操作する環境を目指す

2. **数理的秩序の音楽化**
   - 確率論、フラクタル、セル・オートマトンを「装飾」ではなく「骨格」として統合
   - → 乱数ではなく「確率の設計」としてパラメータを扱う

3. **聴覚体験の純化**
   - 伝統的な音楽の文脈（パフォーマーの崇拝、旋律の情緒性）を解体
   - 純粋な音響エネルギーとしての音楽を提示
   - → リバーブや感情的なメロディへの依存を疑う
